# PART ONE: SYSTEM OVERVIEW

## 1.1 Purpose

This document provides manual verification protocols that serve the same function as automated algorithms. Each protocol is designed to be executable by a human composer with pen, paper, and basic calculation tools.

## 1.2 Protocol Categories

| Category | Function | Execution Frequency |
|----------|----------|---------------------|
| **Calculation Protocols** | Compute numerical values (HQ, density, coefficients) | Per passage or per session |
| **Scanning Protocols** | Search text for specific terms or patterns | Post-composition |
| **Verification Protocols** | Confirm parameters meet specifications | Pre and post-composition |
| **Tracking Protocols** | Log appearances and update registries | Ongoing |
| **Integration Protocols** | Ensure cross-system coherence | Per passage and weekly |

## 1.3 Tools Required

- Calculator (basic arithmetic)
- Word processor with search function
- Printed reference tables (appendices)
- Tracking spreadsheet or ledger
- Highlighters (color-coded by system)
- Page/word count function
