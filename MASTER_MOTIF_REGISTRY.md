# PART EIGHT: SPACING COEFFICIENT MANAGEMENT

## 8.1 Coefficient of Variation Calculation

**Target**: 0.40 - 0.70

**Formula**:
```
CV = Standard Deviation of Intervals / Mean Interval

Where Interval = pages between successive appearances of same motif
```

### Interpretation

| CV Value | Interpretation | Action |
|----------|---------------|--------|
| < 0.40 | Too regular; predictable | Introduce more variation |
| 0.40-0.55 | Slightly regular; acceptable | Minor adjustment if persistent |
| 0.55-0.65 | Optimal range | Maintain |
| 0.65-0.70 | Slightly random; acceptable | Minor adjustment if persistent |
| > 0.70 | Too random; no pattern forming | Regularize spacing |
